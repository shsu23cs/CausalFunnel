About CausalShop

CausalShop is a demo e-commerce storefront built to demonstrate the CausalFunnel user behavior tracking system. Every page view and click interaction on this page is being tracked in real time and displayed in the analytics dashboard.

Navigate between pages, click on products, and add items to cart — then open the CausalFunnel dashboard at http://localhost:8080 to see your session's user journey and heatmap in real time.

🎯
Session Tracking
Every user gets a unique session ID stored in localStorage. All interactions are attached to that session.
🗺️
Heatmaps
Click coordinates are recorded and visualized as an interactive heatmap, showing where users click most.
